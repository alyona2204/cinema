# Стек технологий проекта

Данный проект представляет собой современное React-приложение, разработанное с использованием передовых инструментов и библиотек. Ниже приведено описание используемого стека технологий.

---

## 🧱 Основные технологии

### **React 19**
- Используется версия React (19.2.4)

### **React Router DOM v7**
- Управление маршрутизацией на стороне клиента.

### **TypeScript (~5.9.3)**
- Обеспечивает строгую типизацию, что повышает надёжность кода и упрощает разработку.

---

## ⚙️ Инструменты сборки и разработки

### **Vite (^8.0.1)**
- Современный сборщик и сервер разработки с мгновенной загрузкой HMR (Hot Module Replacement).
- Обеспечивает быстрый старт, высокую производительность и отличную интеграцию с React и TypeScript.

---

## 🧼 Качество кода и линтинг

### **ESLint (^9.39.4)**
- Анализатор кода для выявления ошибок и обеспечения единообразия стиля.

### **Prettier (3.8.1)**
- Форматирование кода для единообразного стиля.
- Интегрирован с ESLint для бесконфликтной работы.

---

## 💾 Дополнительные зависимости

### **Bootstrap ^5.3.8**
- CSS-фреймворк для адаптивного дизайна и компонентов интерфейса.

### **moment ^2.30.1**
- Библиотека для работы с датами и временем.

### **qrcode.react ^4.2.0**
- Компонент для генерации QR-кодов прямо в интерфейсе React.
